This project is a **Python-based Digital Twin** that virtually replicates the behavior of a car engine.  
It simulates real-time engine parameters, performs stress testing, analyzes engine health, and predicts maintenance needs ‚Äî all **without using physical sensors or hardware**.

The goal of this project is to explore the **Digital Twin concept** and understand how virtual models can mirror real-world systems for testing, diagnostics, and optimization.

## üîß Features

- Real-time simulation of engine parameters:
  - RPM, Speed, Engine Load
  - Coolant & Oil Temperature

- Multiple test scenarios:
  - Idle
  - Acceleration
  - Steady cruise
  - High-load (stress test)
- Engine health analysis with system scores
- Predictive maintenance estimation based on mileage
- Noise-injected sensor data for realism
- Export of simulation data to JSON format
